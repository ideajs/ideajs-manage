vue.js配置文件修改

1.解决npm run build打包后运行报错的问题
错误信息：Uncaught SyntaxError: Unexpected token <
解决办法：
  打开文件：config/index.js
  修改内容：
            build: {
              assetsPublicPath: '/' // 修改为：assetsPublicPath: './'
            }

2.package.json文件修改：

"scripts": {
  "dev": "node build/dev-server.js",
  "start": "node build/dev-server.js",
  "develop": "cross-env BABEL_ENV=dev node build/build.js",
  "test": "cross-env BABEL_ENV=test node build/build.js",
  "build": "cross-env BABEL_ENV=prod node build/build.js",
  "unit": "cross-env BABEL_ENV=test karma start test/unit/karma.conf.js --single-run",
  "lint": "eslint --ext .js,.vue src test/unit/specs",
  "fix-memory-limit": "cross-env LIMIT=40096 increase-memory-limit"
}

注意：
  本地运行：  npm run dev
  测试打包：  npm run test
  生产打包：  npm run build
