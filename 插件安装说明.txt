插件安装说明

vue.js项目安装插件列表：

1. iView：HTML5前端-PC客户端插件库
  npm install iview --save

2. VUX：HTML5前端-移动客户端插件库
  npm install vux --save
  npm install vux-loader --save
  npm install less less-loader --save-dev
  npm install yaml-loader --save-dev

3. vue-resource：http请求方法：post/get及跨域请求支持
  npm install --save vue-resource

4. stylus：css样式预处理器
  npm install stylus --save-dev
  npm install stylus-loader --save-dev

5. vuex：Vuex 是 store（仓库）
  npm install vuex --save

6. vuex-router-sync：把vue-router的数据状态同步到vuex的state中
  npm install vuex-router-sync --save

7. base64 + md5：数据加密
  npm install --save js-base64
  npm install --save js-md5

8. Echarts：图表：
  npm install echarts -S

9. PDFJS：文档阅读：
  npm isntall --save pdfjs-dist

10. vue-touch：手机端左右滑屏插件
  npm install vue-touch@next --save

11. vue-qr：生成中间带图片的二维码
  npm install vue-qr --save

12. vue-cropper：组件实现手机端头像图片裁切上传
  npm install vue-cropper

13. increase-memory-limit：增加node服务器内存限制
  npm install increase-memory-limit --save-dev

14. mysql：连接mysql数据库工具
  npm install mysql --save

15. express：Node.js Web 应用程序开发框架
  npm install express --save

16. body-parser：express的中间件
  npm install body-parser --save

17. cross-env 运行跨平台设置和使用环境变量的脚本
  npm install --save-dev cross-env

18. vconsole 移动端网页调试神器
  npm install vconsole --save
